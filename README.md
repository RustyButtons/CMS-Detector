# CMS-Detector
A Python script to detect the CMS of a specific website.

## Installation
```bash
$ pip3 install virtualenv
$ virtualenv -p python3 .venv
$ source .venv/bin/activate
$ pip3 install -r requirements.txt
$ python3 cms_detector.py -h
```
